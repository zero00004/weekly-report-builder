🚀 快速开始

安装 Python（3.8+）
pip install python-docx requests
git clone https://github.com/zero00004/weekly-report-builder.git
cd weekly-report-builder && python report_builder.py --demo

就是这么简单！运行后会在当前目录生成 周报_日期.docx。

📝 使用示例

把每天的工作记录下来，然后交给 AI：

周一：开会讨论项目进度，修改了登录页面的bug
周二：完成了用户管理模块的开发，写了单元测试
周三：和产品经理对接新需求，评估了工作量
周四：解决了分页查询的性能问题（加了索引速度提升10倍）
周五：部署了新版本到测试环境

↓ AI 自动整理成周报 ↓

本周工作内容：
1. 完成用户管理模块开发及单元测试覆盖
2. 修复登录页面 Bug
3. 解决分页查询性能问题（优化后速度提升 10 倍）
4. 完成新版本测试环境部署

重点项目/成果：
- 性能优化：通过数据库索引优化，查询速度提升 10 倍

下周计划：
- 继续推进搜索功能开发
- 编写技术设计文档
