# 一键周报生成器

AI 整理工作记录，一键生成专业 DOCX 周报。**永久免费**，零成本运行。

## 用法

```bash
# 安装依赖
pip install python-docx requests

# 生成示例周报
python report_builder.py --demo

# 从文件读取工作记录
python report_builder.py --file notes.txt

# 交互式输入
python report_builder.py
```

## 功能

- AI 智能整理工作记录 → 结构化周报
- 专业 DOCX 排版，可直接打印/提交
- 支持工作内容、重点项目、问题解决、下周计划
- 永久免费，使用智谱 GLM-4-Flash 模型

## 示例

输入你的每日工作记录：

```
周一：开会讨论项目进度，修改了登录页面的bug
周二：完成了用户管理模块的开发
周三：和产品经理对接新需求，评估了工作量
```

AI 自动整理后生成专业周报。
